How It Works

Host User

Host User is the head of the fraternity who will maintain the host module.He/she is responsible for assigning guards to the particular event.The host will be creating the events for that particular fraternity.The host is responsible for maintaining the members of that fraternity.

Guard User

Guard user is the assigned guard.Host assigns guards for the particular event.He has to stay sober for that event.Guard receives a notification in case of emergency.He has to react to that alert raised so that issue can be resolved.

Basic User

Basic users are the members who are attending the party.They can raise alert in case of emergency.They can edit their profiles through web application.
