How It Works

Host User

Host User is the head of the fraternity who will maintain the host module.He/she is responsible for assigning guards to particular event.Host will be creating the events for that particular fraternity.Host is responsible for maintaining the members of that fraternity.

Guard User

Guard user is the basic user.Host assigns guards for particular event.He has to stay sober for that event.Generally Guard users are Males.Guard receives a notification in case of emergency.He has to react to that alert raised so that issue can be resolved.

Basic User

Basic Users are the members of fraternity.They are generally women.They can raise alert in case of emergency.They can edit their profiles through web aplication.
